Designsystem · Mai 2026

Et univers for nysgjerrige hoder.

Et levende stilkart for Lektor Tronrud — verktøyene, tavlevisningene, og sidene jeg bygger for elever, lærere og alle som er sultne på å lære. Bygget på det som allerede står sterkt i klasserommet: tydelig typografi, runde kanter og skygger som faller som krittstøv.

En palett hentet rett ut av logoen

04 / Skygger

Krittstøv, ikke bokeh

Skyggene har null blur. De faller som krittstøv ned-og-til-høyre, som om kortene er klippet ut og limt på papir. Ingen myke gradienter, ingen Apple-aktige glow.

05 / Knapper

Alle tilstander, alle størrelser

Hver knapp har en kant, en hard skygge, og en bevegelse oppover-til-venstre på hover. Trykker du, klapper den ned mot papiret.

07 / Kort

Seks varianter, ett DNA

Alle kort har den samme harde skyggen og 2-piksels kanten. Det er innholdet som varierer — ikke beholderen.

.card · standard

Standard kort

Hverdagskortet. Hvit flate, 2px kant, hard skygge. 80 % av alt på sidene mine bor i dette kortet.

.card.--soft

Mykt kort

Tynnere ramme, ingen skygge. For lister og innhold som ikke skal hoppe ut av siden.

.card.--ink

Blekk-kort

Mørk variant for nøkkelinformasjon — som tema-kortet på tavlen, eller hero-seksjoner.

.card.--accent-top

Topp-aksent

Fargekoder kortet uten å fylle hele bakgrunnen. Brukes til tema-grupperinger.

Nivå 1
.card.--task

Oppgavekort

Tavlevariant. Fargebånd på toppen + tilhørende skillelinje. Brukes på Klasseromsvisning.

Nivå 2

Oppgavekort · oransje

Samme komponent, andre token. Bytt fargen for å markere mellomnivå.

12 / Tokens i kode

Kopier & lim inn

Bruk disse variablene som limen mellom alle nye sider. Limer du dem inn øverst i en ny HTML, snakker den allerede systemets språk.

CSS Custom Properties
/* === Lektor Tronrud · designtokens === */ :root { /* Brand neutrals */ --space-900: #0E1A38; --slate-700: #2D3748; --slate-500: #5B6478; --paper: #F3F4F2; --surface: #FFFFFF; /* Cosmos accents */ --nova-yellow: #F4C430; --comet-orange: #F5A623; --moss-green: #8BC34A; --planet-teal: #22C8C8; --aurora-blue: #00BCD4; --berry-pink: #E91E63; /* Type */ --font-display: 'Rubik', system-ui, sans-serif; --font-body: 'Atkinson Hyperlegible', system-ui, sans-serif; /* Spacing — 4px base */ --s-1: 4px; --s-2: 8px; --s-3: 12px; --s-4: 16px; --s-6: 24px; --s-8: 32px; /* Radii */ --r-md: 10px; --r-lg: 14px; --r-pill: 9999px; /* Signature shadows — hard, no blur */ --shadow-1: 2px 2px 0 rgba(14,26,56,0.20); --shadow-2: 4px 5px 0 rgba(14,26,56,0.28); --shadow-3: 6px 7px 0 rgba(14,26,56,0.32); }
